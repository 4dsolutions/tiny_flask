Here's the API:

(returns JSON)

/api/elements?elem=H
/api/glossary?term=AJAX

/api/elements?elem=all
/api/glossary?term=all

More options:

(returns HTML)

/elements
/elements/H
/elements/all

with parallel API for /glossary

/glossary
/glossary/.NET
/glossary/all

and for /polyhedra (under development)

/shapes
/shapes/icosa
/shapes/all


Polyhedrons: Platonics + Dual Combos such as Rhombic Triacontahedron


The JSON API (for scripts and bots) applies to all of these as well


POST methods implemented for submitting data remotely (example script), for adding to either database.
